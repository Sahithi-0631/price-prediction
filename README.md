 ## Uber Fare Prediction App
 ## Overview
This project predicts Uber fare prices using Machine Learning (Linear Regression) combined with real-world pricing logic.
## Features
- Predicts fare based on distance, passengers, and time
- Night surge pricing (10 PM – 6 AM)
- Peak hour pricing (8–10 AM, 5–8 PM)
- Streamlit web application
- Real-time predictions
